In order to use this Jupyter notbook:

1) Install Anaconda (Python3) along PyTorch from:
    https://www.anaconda.com/distribution/
    https://pytorch.org

2) Put the /path/to/your/anaconda/bin directory in your PATH,
   such that your machine runs the Anaconda Python version,
   when you type python in your commandline.

3) In your terminal, run:
    $ jupyter notebook


4) Navigate to the corresponding ipynb file and run cells:
    e.g. Cell -> Run All
